# 세션 8 시작 프롬프트

CLAUDE.md와 PROGRESS.md를 읽고 세션 프로토콜에 따라 시작해라.
이번은 세션 8이다: Phase 15(Non-Developer Web UI).
필독 명세: docs/spec/02, 05(자문 화면 요건), 10.

구현 범위: React 18 + TypeScript + Vite 프론트엔드(Docker Compose 내 별도 컨테이너),
로그인, 프로젝트 목록, 대화 화면(채팅 + 문서 업로드 + 질문 카드), 승인 화면(승인 카드),
진행 상태 화면(WebSocket 실시간), 결과 확인·다운로드 화면(기능 확인 체크리스트),
히스토리 화면, 전문가 자문 화면(패키지 다운로드/복사, 질문별 답변 입력).
코드·Diff·기술 용어를 기본 노출하지 않는 UI 원칙을 지켜라.

종료 조건: 브라우저에서 로그인 → 기획안 입력 → 승인 → 상태 확인 → 다운로드 흐름 수동 확인,
frontend 테스트(vitest) 통과.
종료 시 PROGRESS.md 갱신 및 커밋.
